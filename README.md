# Gladiators_game - это компъютерная игра, где игроки рисуют своих "гладиаторов" и соревнуются у кого получилось лучше.
В нашей игре ограничений почти нет, можно дать фантазии поиграть и создать самого красивого персонажа, также можно добавить тени с помощю эффекта и прибавить объемности с помощью иллюзии
